# 📻 Radio API
**API для загрузки музыки в MP.**
*Для начала нужно скачать сам репозиторий и удалить всё кроме папки с данными мода.*
*Затем вы можете изменить название музыки в файле Data [{"sound":{"AssetId":"Музыка.wav"
И саму название текстуры (формата png/jpg)
"mainTexture":{"AssetId":"Радио.png"*
**Затем вы можете добавить свою музыку обязательно в формате wav.**
*(Для этого есть всякие конверты из mp3. В wav и др.*
*Дальше заменяте музыку на свою и сжимаете до zip архива и изменяете формат из **zip** в **melmod**.
**Кидайте в папку** *Телефон/Android/data/com.studio27.MelonPlayground/files/Mods*
*Дальше заходим в MP спавнии радио и включаем музыку.*
